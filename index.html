Short Description

Cave Escape is an engaging 2D game that challenges players to navigate through a perilous cave filled with obstacles, requiring precise timing and skill to survive. With its captivating gameplay and retry mechanics, it keeps players motivated to achieve high scores.

Features

  • Unique gameplay mechanics requiring quick reflexes.
  • Challenging obstacles that increase in difficulty.
  • Retry and menu buttons for seamless gameplay flow.
  • Dynamic level transitions and animations.

Development Tools

  • Unity Game Engine
  • C# Programming Language
  • TextMesh Pro for UI elements
  • Photoshop for sprite creation
  • Audacity for sound effects
